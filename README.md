# Md Sahil Ahmad - Personal Portfolio

This is a modern, highly interactive personal portfolio website for Md Sahil Ahmad, a multidisciplinary creative professional specializing in digital marketing, video editing, photography, videography, and graphic design. Built with Next.js, this portfolio showcases his work, skills, and provides multiple avenues for client engagement.

## Features

*   **Dynamic Hero Section**: Engaging introduction with animated text, rotating skills display, and parallax effects.
*   **AI Assistant**: An interactive chat assistant providing quick answers about services, portfolio, and contact information, with chat history saved locally.
*   **Smooth Scrolling**: Enhanced user experience with smooth scrolling powered by the Lenis library.
*   **Dark/Light Theme Toggle**: Seamless switching between dark and light modes for personalized viewing.
*   **Background Video**: An immersive background video with theme-aware overlays, ensuring a cinematic feel across devices (with MP4 fallback for iOS compatibility).
*   **About Section**: Detailed professional overview with animated statistics.
*   **Services Showcase**: Comprehensive display of specialized services with engaging visuals.
*   **Filterable Portfolio**: A curated collection of projects, filterable by category, demonstrating diverse expertise.
*   **Skills Overview**: Visual representation of technical proficiencies and core competencies with animated progress bars.
*   **Contact Form & Modals**: Multiple contact points, including a dedicated contact section and a modal inquiry form that integrates with WhatsApp and email.
*   **Floating WhatsApp Button**: Quick and easy direct communication via WhatsApp.
*   **Responsive Design**: Optimized for a seamless experience across all devices (desktop, tablet, mobile).
*   **Animations**: Extensive use of `framer-motion` for fluid and captivating UI animations.

## Technologies Used

*   **Next.js**: React framework for production.
*   **React**: JavaScript library for building user interfaces.
*   **Tailwind CSS**: A utility-first CSS framework for rapid UI development.
*   **Framer Motion**: A production-ready motion library for React.
*   **Next-Themes**: For managing and switching between dark and light themes.
*   **Lenis**: A smooth scrolling utility for web pages.
*   **Lucide React**: A collection of beautiful and customizable open-source icons.

## Getting Started

Follow these steps to set up the project locally.

### Prerequisites

*   Node.js (v18 or later recommended)
*   npm or yarn

### Installation

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd nextjs-app
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```
3.  **Run the development server:**
    ```bash
    npm run dev
    # or
    yarn dev
    ```
    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Building for Production

```bash
npm run build
# or
yarn build
```
This command builds the application for production to the `.next` folder.
